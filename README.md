# PostXeno E-Commerce Platform

PostXeno is a full-stack e-commerce web application designed to simulate a real-world online store. It includes product management, secure checkout, and an admin dashboard for managing store operations.

---

## 🚀 Features

- User authentication (login/register)
- Product listing and browsing
- Admin dashboard for product management
  - Create products
  - Update products
  - Delete products
- Secure checkout integration (Stripe)
- Order creation and tracking
- RESTful API architecture

---

## 🛠️ Tech Stack

### Frontend
- React
- Axios
- CSS / Custom styling

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

### Payments
- Stripe API

### Tools & Environment
- VS Code
- Git & GitHub

---

## 🧱 Architecture Overview

The application follows a typical full-stack architecture:

- **Frontend (React)** handles UI and API requests
- **Backend (Express)** handles business logic and routing
- **Database (MongoDB)** stores users, products, and orders
